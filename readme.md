# DMT
Date Memory Top collector and analyzer

## Usage 
```
usage: analyzer.py [-h] -f F [-o O] [-t {meminfo,res,shr,cpu,mem}] [-p P]

DMT analyzer

options:
  -h, --help            show this help message and exit
  -f F                  infile
  -o O                  outfile
  -t {meminfo,res,shr,cpu,mem}
                        Plot Types
  -p P                  PIDs
```
